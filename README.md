# Web Studio

A modern, responsive landing page for a digital agency. The site presents services, team members, portfolio items, and contact options in a clean and structured layout.

## Features

- **Hero Section** – Prominent headline, tagline, and “Order Service” call-to-action.
- **Advantages** – Highlights key strengths: strategy, punctuality, diligence, and technology.
- **Team** – Profiles of four specialists (Product Designer, Frontend Developer, Marketing, UI Designer) with photos.
- **Portfolio** – Gallery of completed projects, including app designs and marketing cases.
- **Contact Form** – Fields for name, phone, email, and message; includes a privacy policy agreement.
- **Footer** – Contact details such as email and phone number.

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- Fully responsive layout for desktop and mobile

## Usage

1. Clone or download the repository.  
2. Modify `index.html` to update text, imagery, and contact information.  
3. Adjust existing CSS or add new styles to match your preferred design.  
4. Deploy using any static hosting provider (GitHub Pages, Netlify, Vercel, etc.).
